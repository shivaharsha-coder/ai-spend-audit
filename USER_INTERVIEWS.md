# User Interviews

## User 1 (Student)
Problem: Uses multiple AI tools but doesn’t track cost  
Need: Simple way to know if spending is too high  

## User 2 (Freelancer)
Problem: Pays for tools but unsure if they are worth the cost  
Need: Suggestions to reduce spending  

## User 3 (Small Team)
Problem: Shared subscriptions without tracking usage  
Need: Visibility on spending and usage  

## Key Insights
- Users want simple and quick feedback
- Users care about saving money
- Dashboard view improves understanding
