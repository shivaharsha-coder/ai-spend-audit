# Reflection

## What I Built
I built an AI Spend Audit application that helps users analyze and optimize their AI tool spending.

## Why This Project Matters
With increasing usage of AI tools, users often subscribe to multiple services without tracking costs.  
This project helps bring clarity and cost awareness.

## What I Learned
- Building apps using Next.js
- Managing state using React
- Implementing login using localStorage
- Improving UI/UX design
- Writing structured product documentation

## Challenges
- Managing login flow
- Designing a clean UI
- Writing clear documentation

## How I Solved Them
- Used simple logic and structure
- Focused on user experience
- Broke problems into small steps

## Future Improvements
- Add charts and visual analytics
- Store audit history
- Add backend and database
- Implement real authentication
